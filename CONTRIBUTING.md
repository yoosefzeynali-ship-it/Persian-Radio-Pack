# Contributing to Persian Radio Pack

First of all, thank you for your interest in contributing to the Persian Radio Pack! ❤️

Whether you want to report a bug, suggest a new radio station, or improve the project, your help is always appreciated.

---

## Ways to Contribute

You can contribute by:

- Reporting bugs
- Suggesting new Persian radio stations
- Reporting broken or outdated stream URLs
- Improving documentation
- Submitting Pull Requests
- Suggesting new features

---

## Reporting Issues

Before opening a new issue:

- Make sure the issue has not already been reported.
- Include as much information as possible.
- Describe how the problem can be reproduced.
- Include screenshots if applicable.

---

## Suggesting a New Radio Station

When suggesting a new station, please include:

- Radio Name
- Stream URL
- Genre
- Bitrate (if known)
- Website (optional)

Example:

Name: Example Radio

Genre: Pop

Stream URL:
https://example.com/live.mp3

Bitrate:
128 kbps

---

## Reporting Broken Streams

If a radio station no longer works, please provide:

- Station name
- Current stream URL
- Correct stream URL (if available)

---

## Pull Requests

Before submitting a Pull Request:

- Keep changes focused and small.
- Follow the existing project structure.
- Test your changes before submitting.
- Update documentation if necessary.

---

## Code Style

Please keep:

- Clear file names
- Consistent formatting
- Meaningful commit messages

---

## Support

If you have questions or suggestions, feel free to open an Issue or start a Discussion on GitHub.

---

## Thank You

Every contribution, no matter how small, helps improve the Persian Radio Pack for the ETS2 and ATS community.

Thank you for supporting the project! 🚛❤️
