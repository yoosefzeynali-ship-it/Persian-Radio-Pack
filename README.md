# Persian Radio Pack

Persian Radio Pack for Euro Truck Simulator 2 and American Truck Simulator.

## Features

- 63 Persian Radio Stations
- Updated Stream URLs
- ETS2 Compatible
- ATS Compatible
- TruckersMP Compatible

## Installation

Copy `live_streams.sii` to:

Documents/Euro Truck Simulator 2/

or

Documents/American Truck Simulator/
