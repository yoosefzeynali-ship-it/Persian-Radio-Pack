# 🎙 Persian Radio Pack

The largest collection of Persian Internet radio stations for **Euro Truck Simulator 2** and **American Truck Simulator**.

> Listen to your favorite Persian radio stations while driving across Europe and America.

---

## ✨ Features

- 📻 63 Persian radio stations
- 🔄 Updated streaming URLs
- 🚚 Compatible with Euro Truck Simulator 2
- 🇺🇸 Compatible with American Truck Simulator
- 🌐 Works with TruckersMP
- ⚡ Easy installation
- 🆓 Free and open-source

---

## 📥 Installation

1. Download the latest `live_streams.sii`.
2. Copy it to:

### Euro Truck Simulator 2

```
Documents/Euro Truck Simulator 2/
```

### American Truck Simulator

```
Documents/American Truck Simulator/
```

3. Launch the game.
4. Enjoy your favorite Persian radio stations!
