# Installation Guide

## Euro Truck Simulator 2

1. Download the latest Persian Radio Pack.
2. Backup your current `live_streams.sii`.
3. Copy the new `live_streams.sii` into:

Documents/Euro Truck Simulator 2/

or simply install it using Persian Radio Manager.

---

## American Truck Simulator

2. Backup your current `live_streams.sii`.
