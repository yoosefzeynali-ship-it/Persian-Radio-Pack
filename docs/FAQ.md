# Frequently Asked Questions (FAQ)

## The radio does not play inside the game.

- Make sure your internet connection is working.
- Verify that the streaming server is online.
- Restart the game after installing a new Radio Pack.
- Check whether the radio station is temporarily unavailable.

---

## Where is the radio file located?

### Euro Truck Simulator 2

Documents/Euro Truck Simulator 2/live_streams.sii

### American Truck Simulator

Documents/American Truck Simulator/live_streams.sii

---

## Is it safe to install?

Yes.

Persian Radio Manager automatically creates a backup of your current
`live_streams.sii` file before replacing it.

---

## How can I restore my previous radio list?

Open Persian Radio Manager and click **Restore Backup**.

The latest backup will be restored automatically.

---

## How do I update to the latest radio pack?

Simply click **Install Radio Pack** inside Persian Radio Manager.

The application automatically downloads and installs the latest release from GitHub.

---

## Which games are supported?

- Euro Truck Simulator 2
- American Truck Simulator

---

## Do I need to download the Radio Pack manually?

No.

Persian Radio Manager downloads the latest version directly from GitHub.
