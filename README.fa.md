# مجموعه رادیوهای فارسی برای Euro Truck Simulator 2 و American Truck Simulator

**Persian Radio Pack** مجموعه‌ای از رادیوهای اینترنتی فارسی‌زبان است که برای بازی‌های **Euro Truck Simulator 2** و **American Truck Simulator** تهیه شده است.

هدف این پروژه، فراهم کردن تجربه‌ای بهتر برای رانندگان مجازی است تا هنگام رانندگی بتوانند به رادیوهای فارسی مورد علاقه خود گوش دهند.

---

# ✨ ویژگی‌ها

* 📻 بیش از ۶۳ ایستگاه رادیویی فارسی‌زبان
* 🔄 لینک‌های استریم به‌روزرسانی‌شده
* 🚚 سازگار با Euro Truck Simulator 2
* 🇺🇸 سازگار با American Truck Simulator
* 🌐 قابل استفاده در TruckersMP
* ⚡ نصب آسان
* 🆓 رایگان و متن‌باز

# 📥 آموزش نصب

## Euro Truck Simulator 2

فایل `live_streams.sii` را در مسیر زیر قرار دهید:

```
Documents/Euro Truck Simulator 2/
```

## American Truck Simulator

فایل `live_streams.sii` را در مسیر زیر قرار دهید:

```
Documents/American Truck Simulator/
```

سپس بازی را اجرا کرده و از قسمت **Radio Player** رادیوی مورد نظر خود را انتخاب کنید.
